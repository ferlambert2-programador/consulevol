# ClinicEvol

Asistente de evoluciones clínicas con IA para médicos.

## Funcionalidades

- 6 tipos de documento: Evolución UTI, Evolución Clínica Médica, Ingreso UTI, Alta UTI, Ingreso Clínica Médica, Alta Clínica Médica
- Carga de PDF de laboratorio (AVlab) con extracción automática
- Carga de PDF de informes de imágenes (Centro Sinclair)
- Fotos de monitor hemodinámico, respirador e informes impresos
- Dictado por voz con transcripción automática (Whisper)
- Generación de evolución narrativa con IA (Claude)
- Botón copiar para pegar en el sistema de la clínica
